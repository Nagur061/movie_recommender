# Movie Recommendation System 🎬

This is a content-based movie recommendation system built using Python and Machine Learning.

## 🔹 Features
- Recommends similar movies based on input
- Uses cosine similarity
- Combines genres, keywords, cast, and director

## 🔹 Technologies Used
- Python
- Pandas
- Scikit-learn

## 🔹 How it Works
1. Data preprocessing
2. Feature engineering (tags)
3. Text vectorization using CountVectorizer
4. Similarity calculation using cosine similarity

## 🔹 Example
Input: Avatar  
Output: List of similar movies

## 🔹 Dataset
TMDB 5000 Movie Dataset (Kaggle)
